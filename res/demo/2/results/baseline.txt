Q: List each company and its services/cloud contribution.
A: Here’s a list of some major cloud service providers and their contributions to the cloud computing landscape:

1. **Amazon Web Services (AWS)**
   - **Services**: Infrastructure as a Service (IaaS), Platform as a Service (PaaS), Software as a Service (SaaS)
   - **Contributions**: Offers a wide range of services including computing power (EC2), storage (S3), databases (RDS), machine learning (SageMaker), and serverless computing (Lambda).

2. **Microsoft Azure**
   - **Services**: IaaS, PaaS, SaaS
   - **Contributions**: Provides services such as virtual machines, app services, Azure SQL Database, Azure Functions, and Azure DevOps, along with strong integration with Microsoft products.

3. **Google Cloud Platform (GCP)**
   - **Services**: IaaS, PaaS, SaaS
   - **Contributions**: Offers services like Google Compute Engine, Google Kubernetes Engine, BigQuery for data analytics, and Firebase for mobile app development.

4. **IBM Cloud**
   - **Services**: IaaS, PaaS, SaaS
   - **Contributions**: Provides cloud computing services including IBM Watson for AI, IBM Cloud Kubernetes Service, and various blockchain and data services.

5. **Oracle Cloud**
   - **Services**: IaaS, PaaS, SaaS
   - **Contributions**: Focuses on database services, offering Oracle Database Cloud, Oracle Cloud Applications (SaaS), and cloud infrastructure services.

6. **Alibaba Cloud**
   - **Services**: IaaS, PaaS, SaaS
   - **Contributions**: Provides a range of services including Elastic Compute Service (ECS), ApsaraDB for databases, and machine learning services.

7. **Salesforce**
   - **Services**: SaaS
   - **Contributions**: Primarily known for its customer relationship management (CRM) software, Salesforce also offers a platform for building applications (Salesforce Platform).

8. **DigitalOcean**
   - **Services**: IaaS, PaaS
   - **Contributions**: Focuses on simplicity and developer-friendly services, offering Droplets (virtual machines), Kubernetes, and managed databases.

9. **VMware Cloud**
   - **Services**: IaaS, PaaS
   - **Contributions**: Provides virtualization and cloud infrastructure services, including VMware Cloud on AWS and VMware Tanzu for Kubernetes.

10. **Linode**
    - **Services**: IaaS
    - **Contributions**: Offers cloud hosting services with a focus on simplicity and cost-effectiveness, providing virtual servers and block storage.

11. **Rackspace Technology**
    - **Services**: Managed cloud services
    - **Contributions**: Provides managed services across various cloud platforms, including AWS, Azure, and GCP, focusing on cloud migration and management.

12. **Heroku**
    - **Services**: PaaS
    - **Contributions**: A platform for building, running, and operating applications entirely in the cloud, known for its ease of use and integration with various programming languages.

This list is not exhaustive, as there are many other cloud service providers and niche players in the market, but these are some of the most prominent companies and their contributions to cloud computing.
